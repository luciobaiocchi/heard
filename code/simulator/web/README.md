# HEARD — 3D Web Visualizer

Live replay of the HEARD simulation on a 3D globe (MapLibre GL JS).
Devices glide along the real GPX trail over real 3D terrain, leaving glowing
path trails behind them. LoRa link arcs flash REQ/WAIT/POS exchanges in real
time, with range domes and a protocol-state sidebar.
Markers are positioned by MapLibre's native terrain projection (no manual offsets).

---

## Quick start (run every time)

```bash
# 1 — go to the simulator directory
cd ~/projects/thesis/code/simulator

# 2 — record the simulation trace (choose a mode)
python3 record.py                    # default: uniform (all devices on path)
python3 record.py --mode normal      # ~1 node briefly leaves the 100 m safe zone for ~5-6 s
python3 record.py --mode hard        # multiple nodes go deep off-path, trigger OUT_PATH for ~15 s

# optional: longer run
python3 record.py --mode hard --ticks 3600   # 6 minutes of sim time

# 3 — serve the web viewer
cd web
python3 -m http.server 8000

# 4 — open the browser
open http://localhost:8000
```

Use the **timeline bar at the bottom** to play / pause / scrub / change speed.

The green dashed corridor around the trail marks the **100 m safe zone** (the
firmware's `Path(maxDistance=100)` threshold). Devices that stray outside it
turn **red** with `⚠ OUT`, and the *Off-path* row in the sidebar tracks them.

```bash
cp heard_sim*.so ..
```

## 3D terrain — MapTiler key (optional)

3D terrain loads automatically using free Terrarium DEM tiles.
A free MapTiler API key unlocks higher-quality outdoor/topo/satellite base maps.
If you want to use it:

1. Sign up at <https://maptiler.com> → **API keys**.
2. Copy your key.
3. Edit `web/config.js`:

```js
window.MAPTILER_KEY = "your_key_here";
```
